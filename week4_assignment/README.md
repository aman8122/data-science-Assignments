# 📘 CIFAR-10 Image Classification Project (Week 4)

An educational Deep Learning project comparing the performance, architectural constraints, and learning behaviors of **Artificial Neural Networks (ANN)** versus **Convolutional Neural Networks (CNN)** using the CIFAR-10 dataset.

## 🧠 Project Overview
- **Dataset:** CIFAR-10 (60,000 32x32 color images across 10 classes)
- **Goal:** Classify images and analyze why spatial structures require convolutional pipelines.
- **Key Concepts Covered:** Spatial Loss in ANNs, Weight Sharing in CNNs, Regularization (Dropout & Batch Normalization), and Data Augmentation.
